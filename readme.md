# EDOO - Estrutura de Dados Orientada a Objetos 💻

Este repositório contém meus estudos, exercícios e atividades práticas desenvolvidas durante a disciplina de **Estrutura de Dados Orientada a Objetos (EDOO)** no curso de Bacharelado em Inteligência Artificial da **UFPE**.

O foco principal é o domínio da linguagem **C++** e a implementação de estruturas de dados clássicas, utilizando como base teórica e prática o livro de Peter Prinz e Ulla Kirch-Prinz.

## 📚 Referência Principal
* **Livro:** *C++: Guia Completo de Programação* (Ulla Kirch-Prinz & Peter Prinz).
* **Conteúdo:** Fundamentos de C++, POO, Gerenciamento de Memória, Ponteiros...

## 📂 Estrutura do Repositório

O projeto está organizado da seguinte forma:

* **`/Atividades práticas c++`**: Resolução das listas de exercícios e atividades práticas (APs) da disciplina.
    * `AP1`, `AP2`, `AP3` `...`: Exercícios de fixação e avaliações.
* **`/livro`**: Implementações baseadas nos capítulos do livro do Kirch-Prinz.
    * `Códigos livro`: Exemplos de sintaxe e conceitos extraídos das páginas do livro.
    * `Exercícios`: Problemas propostos ao final de cada capítulo.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** C++ (Padrão C++11 ou superior).
* **Ambiente:** VS Code.
* **Compilador:** GCC/G++ (MinGW ou similar).
* **Versionamento:** Git & GitHub.

## 🚀 Como Executar
Para compilar qualquer um dos arquivos `.cpp`, você pode utilizar o terminal:

```bash
# Compilar (Exemplo)
g++ exercise7.cpp -o exercise7
 
# Executar
./exercise7
```
